Customer Forecast Upload (26Q4)

Upload a CSV with columns N-Code, Thickness, Width, Ship-to, 2026-10, 2026-11, 2026-12 to bulk-update Customer Forecast values (MT). Existing values for the same N-code + Thickness + Width + Ship-to + month are overwritten; new rows are added. You can also edit individual values directly under each Grade in the 26Q4 Order Forecast table on the Dashboard.

Consignment Stock Upload

Upload a CSV with columns N-Code, Thickness, Width, Ship-to, Consignment Stock to bulk-update warehouse stock held for consignment customers (MT). This reduces the Net Need shown in the 26Q4 forecast table (Target − Produced − Consignment Stock). Existing values for the same N-code + Thickness + Width + Ship-to are overwritten. You can also edit individual values directly under each Grade in the 26Q4 Order Forecast table on the Dashboard.
